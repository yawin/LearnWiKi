## ✨ What's New

- Wiki Q&A now correctly recognizes when an answer is grounded in your saved pages — fixed a bug where answers that did cite wiki pages were sometimes mistagged as "AI only" and couldn't be saved.
- Follow-up questions like *"the first one"* or *"the last one"* now correctly refer back to pages from the previous turn.
- Scanned / image-based PDFs can now be imported: when text extraction returns garbled output, LearnWiki automatically falls back to on-device OCR (macOS Vision).
- Importing large files no longer freezes the window — the work now runs off the UI thread.

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.9_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.9_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder.
3. Double-click LearnWiki to launch — no first-run security prompt.
4. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).

## 安装说明（macOS）

1. 下载与你的 Mac 对应的 `.dmg` 文件：
   - **Apple Silicon（M1/M2/M3/M4）**：`LearnWiki_0.3.9_aarch64.dmg`
   - **Intel 芯片 Mac**：`LearnWiki_0.3.9_x64.dmg`
2. 双击 `.dmg`，将 **LearnWiki** 拖入 **Applications**（应用程序）文件夹。
3. 双击 LearnWiki 即可打开，**不会再有首次启动的安全提示**。
4. 打开应用后进入 **设置 → AI**，填入 AI 服务商的 API Key（Claude / OpenAI / Gemini）。
