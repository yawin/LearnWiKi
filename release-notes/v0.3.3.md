## ✨ What's New

- Fixed v0.3.2 notarization failure — bundled helper binaries (OCR helper, document converter) are now signed before packaging, so Apple's notary service accepts the build. Downloaded DMG opens with one double-click, no security warning.

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.3_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.3_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder.
3. Double-click LearnWiki to launch — no first-run security prompt.
4. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).
