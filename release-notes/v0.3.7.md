## ✨ What's New

- The in-app auto-update is finally working again. Future versions will install themselves with one click.
- If auto-update isn't possible, the update banner now opens the download page automatically so you can grab the new version manually.
- **⚠️ One-time manual update required**: because the update signing was rebuilt this time, the auto-update will fail for users on v0.3.5 and earlier. Please download the DMG below once manually for this version. From v0.3.7 onward, auto-update will work normally.

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.7_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.7_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder.
3. Double-click LearnWiki to launch — no first-run security prompt.
4. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).
