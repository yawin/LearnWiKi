## ✨ What's New

- Smoother in-app auto-update experience: LearnWiki now silently downloads new versions in the background, then shows a small card in the bottom-right corner asking whether to install. One click installs and relaunches instantly — no waiting for the download.
- Improved screen-reader support for the update card.

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.8_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.8_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder.
3. Double-click LearnWiki to launch — no first-run security prompt.
4. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).

## 安装说明（macOS）

1. 下载与你的 Mac 对应的 `.dmg` 文件：
   - **Apple Silicon（M1/M2/M3/M4）**：`LearnWiki_0.3.8_aarch64.dmg`

2. 双击 `.dmg`，将 **LearnWiki** 拖入 **Applications**（应用程序）文件夹。
3. 双击 LearnWiki 即可打开，**不会再有首次启动的安全提示**。
4. 打开应用后进入 **设置 → AI**，填入 AI 服务商的 API Key（Claude / OpenAI / Gemini）。
