## ✨ What's New

- Fixed Intel Mac build of v0.3.0 failing to publish — Intel Mac users can now download and use LearnWiki again

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.1_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.1_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder.
3. **First launch** — because LearnWiki is not notarized with Apple:
   - **Right-click** LearnWiki in Applications → **Open** → confirm **Open** in the dialog, OR
   - Open **System Settings → Privacy & Security**, scroll to the "LearnWiki was blocked" notice, and click **Open Anyway**.
4. Once approved, double-click to launch normally. You'll only need to do this once.
5. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).

> If you're already on v0.3.0 (Apple Silicon), you don't have to upgrade — this release only fixes the Intel Mac build.

## 安装说明（macOS）

1. 下载与你的 Mac 对应的 `.dmg` 文件：
   - **Apple Silicon（M1/M2/M3/M4）**：`LearnWiki_0.3.1_aarch64.dmg`
   - **Intel 芯片 Mac**：`LearnWiki_0.3.1_x64.dmg`
2. 双击 `.dmg`，将 **LearnWiki** 拖入 **Applications**（应用程序）文件夹。
3. **首次打开**（应用未经 Apple 公证，需手动放行一次）：
   - **右键点击**「应用程序」里的 LearnWiki → 选 **打开** → 在弹窗里再点 **打开**，或
   - 打开 **系统设置 → 隐私与安全性**，滚动到底部看到「LearnWiki 已被阻止」，点 **仍要打开**。
4. 放行之后，以后直接双击即可打开，不会再拦截。
5. 打开应用后进入 **设置 → AI**，填入 AI 服务商的 API Key（Claude / OpenAI / Gemini）。

> 如果你已经装了 v0.3.0（Apple Silicon 版），不需要升级，这次更新只是补上 Intel 版本。
