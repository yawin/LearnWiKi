## ⚠️ Important — Manual Install Recommended

If you're upgrading from **v0.3.9 or earlier**, please **download the DMG manually this time**. The old auto-update logic relied on GitHub's API (rate-limited to 60 requests/hour per IP), which can fail silently — leaving the in-app update banner unable to appear. Once you install v0.3.10, all future versions will auto-update reliably.

## ✨ What's New

- Fixed the "Check for Updates" failure that left the in-app updater silently broken — LearnWiki no longer depends on GitHub's rate-limited API to detect new releases.
- Future versions will auto-update reliably without needing manual download.

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.10_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.10_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder (overwrite the old version).
3. Double-click LearnWiki to launch — no first-run security prompt.
4. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).

## ⚠️ 重要提示 — 这次建议手动安装一次

如果你现在用的是 **v0.3.9 或更早版本**，这次请**手动下载 DMG 安装一次**。原因：旧版本的「检查更新」依赖 GitHub 的 API（每个 IP 每小时只能查 60 次），一旦超出就会静默失败——也就是你最近遇到的「右下角不弹卡片、手动检查更新报错」的情况。**装完 v0.3.10 之后，以后所有版本都会正常自动更新，不需要再手动下载。**

## ✨ 本次更新

- **修复了「检查更新」失效**的老 bug——LearnWiki 不再依赖有速率限制的 GitHub API 来检测新版本。
- 以后所有版本都能可靠地自动更新，不需要再手动下载。
