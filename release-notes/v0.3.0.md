## ✨ What's New

- Added file import — drag in Markdown / TXT / PNG / JPG / GIF / WebP / PDF / Word / PowerPoint and they're auto-converted into your knowledge base (no Python install needed, the document converter is bundled in the app)
- Rebuilt the wiki Q&A as a true conversational copilot — handles time-bound questions ("what did I save today / this week"), no more refusals or fabricated page titles, and stays fast as the knowledge base grows
- Weekly reports now work with every AI provider (OpenAI / Gemini / DeepSeek / Qwen / local models / OAuth login), not just Claude
- Added DeepSeek V4 access through the Qwen (DashScope) provider as well
- Each AI provider now remembers the model you picked, so switching providers no longer resets your choice
- Image deduplication now compares the actual image bytes instead of file path + size, so the same picture saved twice won't pile up
- External links inside Q&A answers now open in your system browser instead of taking over the app window
- More forgiving handling of AI responses — duplicated JSON fields, conversational preambles around JSON, and provider-specific quirks no longer break features
- Upgraded the Anthropic model lineup to the current 4.X family (Opus 4.7 / Sonnet 4.6 / Haiku 4.5)

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `LearnWiki_0.3.0_aarch64.dmg`
   - **Intel Mac**: `LearnWiki_0.3.0_x64.dmg`
2. Double-click the `.dmg` and drag **LearnWiki** into the **Applications** folder.
3. **First launch** — because LearnWiki is not notarized with Apple:
   - **Right-click** LearnWiki in Applications → **Open** → confirm **Open** in the dialog, OR
   - Open **System Settings → Privacy & Security**, scroll to the "LearnWiki was blocked" notice, and click **Open Anyway**.
4. Once approved, double-click to launch normally. You'll only need to do this once.
5. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).

## 安装说明（macOS）

1. 下载与你的 Mac 对应的 `.dmg` 文件：
   - **Apple Silicon（M1/M2/M3/M4）**：`LearnWiki_0.3.0_aarch64.dmg`
   - **Intel 芯片 Mac**：`LearnWiki_0.3.0_x64.dmg`
2. 双击 `.dmg`，将 **LearnWiki** 拖入 **Applications**（应用程序）文件夹。
3. **首次打开**（应用未经 Apple 公证，需手动放行一次）：
   - **右键点击**「应用程序」里的 LearnWiki → 选 **打开** → 在弹窗里再点 **打开**，或
   - 打开 **系统设置 → 隐私与安全性**，滚动到底部看到「LearnWiki 已被阻止」，点 **仍要打开**。
4. 放行之后，以后直接双击即可打开，不会再拦截。
5. 打开应用后进入 **设置 → AI**，填入 AI 服务商的 API Key（Claude / OpenAI / Gemini）。
